61-09 Set a focus subject, then check action-plan. → Focus subject items get urgency boost (urgency reduced by up to 3, min 1).
61-10 Call recommendations for invalid/nonexistent student ID. → Returns 404 error, not a crash.

Data Quality

61-11 Call action-plan for invalid student. → Returns 404, not a crash or empty 200.
61-12 Verify each action-plan item has: subject, topic, mode, urgency, reason, label. → All fields populated in every action item.
61-13 Verify lesson_count and practice_count present in recommendation objects. → Numeric counts reflecting actual student activity.
